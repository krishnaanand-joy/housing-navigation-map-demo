# Google My Maps Import

This demo creates:

```text
outputs/google_maps_import.csv
```

That file can be imported into Google My Maps for a simple private map layer.

## How to import

1. Go to Google My Maps.
2. Create a new map.
3. Click **Import** on a map layer.
4. Upload `outputs/google_maps_import.csv`.
5. Choose `latitude` and `longitude` as location columns.
6. Choose `label` as the title column.

## Privacy warning

Only use synthetic data or approved private data.

Do not upload client names, landlord names, real addresses, phone numbers, emails, or case notes to a public or shared map.
